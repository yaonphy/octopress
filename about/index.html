About Octostrap3

Octostrap3 started because I really wanted to use Bootstrap while posting on my blog and all Octopress themes were based on Bootstrap 2.x.

Contributing

For issues, questions and pull requests check the GitHub repository.

License

Just as Octopress Octostrap3 is under the MIT License.

Note that Twitter Bootstrap, which is included in the theme, is licensed under the Apache License v2.0. This can raise some incompatibilities (notably with the GPLv2, see twbs/bootstrap#2054) so care must be taken when sublicensing.
